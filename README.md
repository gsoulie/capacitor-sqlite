# capacitor-sqlite

Store data in local sqlite database *using capacitor sqlite with ionic*

permission issue : https://github.com/capacitor-community/sqlite/issues/55      

## Installation

````
npm install @capacitor-community/sqlite
npm run build
npx cap sync
npx cap add android
npx cap add ios
````
